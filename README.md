BlogVerse
BlogVerse is a modern, dark-themed blogging platform where users can sign up, log in, create posts, comment, and manage personal settings. 
The platform uses Firebase for authentication and database management, providing a smooth, responsive, and secure user experience.

Project Overview
BlogVerse allows users to interact in a sleek and user-friendly interface. After logging in, users can:
Navigate between Home, Posts, Profile, and Settings
Create and comment on posts
Manage theme, notifications, and privacy settings
Access contact links and social media
The app is designed with a modern dark theme, responsive layout, and interactive UI elements.
 
Features
Authentication: Sign Up, Login, Logout, Forgot Password
Protected Pages: Only accessible after login
Posts & Comments: Users can create posts and comment, stored in Firebase
Profile Page: Displays username and email
Settings: Theme, Notifications, Privacy
Contact & Social Links: Email, Twitter, Facebook, Discord
Modern Dark-themed UI with icons

Tech Stack
Frontend: HTML, CSS, JavaScript
Backend / Database: Firebase Authentication & Firestore
Libraries: Firebase JS SDK, Font Awesome Icons
